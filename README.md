# minture_music(仿网易云小程序版本）
当前最新版本： 1.0.0（发布时间：20210306）

## Introduction
minture_music采用原生wx小程序的方式实现音乐全局播放、 每日推荐、首页推荐、浏览MV、等功能。

## technology stack
基于原生wx的小程序
 fork netease-cloud-muisc-api project  
 
 build a node http interface-serve
